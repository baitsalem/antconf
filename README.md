# AntTrust
Protocole de routage Ad Hoc  basé sur les systèmes multi-agents

Ce code a été testé avec ns-allinone-2.34 sous Linux

Pour intégrer le protocole à ns2, vous devez modifier les fichiers suivants dans ns2:

cmu-trace.h,
ns-agent.tcl,
ns-packet.tcl,
cmu-trace.cc,
ns_tcl.cc,
ns-mobilenode.tcl,
ptypes.cc,
ns-lib.tcl,
packet.h
